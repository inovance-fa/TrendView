Introduction
--------------------------
TrendViewOpenSource is a library based on QCustomPlot which provides Qt/C++ primitives to ease building real time plot UI.

Dependencies
--------------------------
Qt >= 5.11
QCustomPlot (included in this repo)

How to Build
--------------------------
The project supports qmake and CMake build systems. Both are supported by QtCreator, and can be used to build and debug.
